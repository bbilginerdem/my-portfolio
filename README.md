# Modern Portfolio & Blog

A high-performance, premium personal portfolio and blog built with the latest web technologies. This project showcases a clean, responsive design with smooth animations and a robust server-side rendered blog system.

## 🚀 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **UI & Logic**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Linting & Formatting**: [Biome](https://biomejs.dev/)
- **Typography**: [DM Sans](https://fonts.google.com/specimen/DM+Sans) (via Google Fonts)

## ✨ Key Features

- **SSR Blog System**: Dynamic, SEO-friendly blog routes with optimized metadata.
- **Modern UI/UX**: Features glassmorphism, smooth entrance animations, and interactive hover states.
- **Dark Mode**: Fully integrated dark mode with a reliable toggle system.
- **SEO Optimized**: Includes dynamic `sitemap.xml`, `robots.txt`, and server-rendered meta tags.
- **Safe Contact**: Secure contact section with email-copy functionality and social links.
- **Fast Build**: Powered by Turbopack for lightning-fast development and production builds.

## 📂 Component Structure

The project follows a clean, atomic organization:
- `components/layout/`: Persistent shells (Header, Footer, Theme toggle).
- `components/home/`: Specific landing page sections.
- `components/ui/`: Reusable primitive components.

## 🛠️ Getting Started

First, install dependencies:
```bash
pnpm install
```

Run the development server:
```bash
pnpm dev
```

Build for production:
```bash
pnpm build
```
